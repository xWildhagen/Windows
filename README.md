# Windows

## Setup

### Installation

1. (Optional) Log in to account(s)
2. Windows update
3. Winget update

```
winget update --all
winget install git.git
git clone https://github.com/xWildhagen/Windows
```

4. Microsoft Store update
5. Driver update

### Other

1. Windows Security
2. Log in to OneDrive
3. Attach NAS
4. Rearrange taskbar

## Stuff

### Pull changes from GitHub

```
git -C windows pull
git -C windows reset --hard
```
